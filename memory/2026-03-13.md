### Template building
- Word fragmenta texto em múltiplos `<w:r>` runs — substituição simples não funciona para todos os campos
- Campos mais problemáticos: RG (3 runs), "brasileiro, solteiro" (run único mas adjacente a profissão em run separado), cabeçalho do órgão (4-5 runs)
- Solução: Python script com substituições cirúrgicas run a run via string replace no XML
