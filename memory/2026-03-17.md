## 3. LIÇÕES APRENDIDAS

### Python `-c` via Telegram perde indentação
- Telegram strip espaços/indentação quando copia comandos multiline
- Solução definitiva: usar dict comprehension em uma linha só (sem for loop com indentação)
- Formato seguro: `python3 -c "...tudo em uma linha com ponto-e-vírgula..."`
- `cat > arquivo << 'EOF'` também falha pelo mesmo motivo (Telegram remove indentação)
- **Nunca mais enviar scripts multiline via Telegram para rodar via `-c`**
