## LIÇÕES APRENDIDAS

### Operador `LIKE` no ClickUp API
- **Lição:** ClickUp pode remover operadores sem aviso → workflows que dependem de filtros customizados precisam ser monitorados
- **Mitigação:** usar operadores básicos (`=`, `!=`) sempre que possível; evitar `LIKE`
- **Impacto real:** 10 dias de captura de leads perdida (02-12/03)

### jsCode no N8N — encoding de newlines
- Quando código JavaScript é armazenado via API com `\n` literais (escaped), o N8N pode não executar corretamente
- **Sintoma:** código aparentemente correto retorna valores zerados/incorretos
- **Diagnóstico:** verificar se `\n` no jsCode é literal (4 chars `\`, `n`) ou newline real (1 char)
- **Fix:** reescrever o código via UI do N8N ou garantir que a API receba newlines reais

### GCP App em modo Testing — Tokens expiram em ~7 dias
- **Causa raiz** das falhas periódicas de OAuth Google Ads/Sheets
- **Fix permanente:** mover app GCP para modo Production
- **Workaround atual:** re-autenticar manualmente quando token expira

### Múltiplas mudanças de infraestrutura — PROIBIDO simultaneamente
